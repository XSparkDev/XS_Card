# This directory contains certificates for wallet pass generation
# 
# Required files for Apple Wallet:
# - passcert.pem (Pass signing certificate)
# - passkey.pem (Pass signing private key)
# - wwdr.pem (Apple WWDR certificate)
#
# Required files for Google Wallet:
# - google-wallet-service-account.json (Google Cloud service account key)
#
# These files are not committed to Git for security reasons.
# Place your actual certificate files here after setting up Apple Developer and Google Cloud accounts.
